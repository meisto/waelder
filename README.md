## Custom Markdown Specs 

**Supported:**
- Italic (\*sth\*)
- Bold (\*\*sth\*\*)
- Bold and italic (\*\*\*sth\*\*\*)
- Headers (\# sth, \#\# sth, ...)
- Ordered List (1. sth, 2.sth, ...)
- Unordered List (- sth, -sth, ...)
- Linebreak ("  \\n")
